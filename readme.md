# Employee Management System (EMS)

## Overview
The Employee Management System (EMS) is a web-based application designed to manage core human resource operations within an organization. It centralizes employee data, streamlines administrative workflows, and supports decision-making across HR, management, and administration.

## Core Features
- Employee profiles (personal details, contact info, job titles, departments)
- Attendance tracking and leave management
- Payroll, salary calculations, taxes, bonuses, and benefits
- Recruitment management (job postings, applications, interviews)
- Performance reviews (KPIs, evaluations, promotions, disciplinary actions)
- Role-based user access (Admin, HR, Manager)

## Tech Stack
- **Frontend:** HTML, Tailwind CSS
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache (XAMPP)

## Project Goal
To provide a modular, scalable HR management platform suitable for small to medium-sized organizations while maintaining clarity, simplicity, and data integrity.
